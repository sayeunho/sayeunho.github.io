### 1. Loops (반복문)

```javascript
let i = 3;
while ( i > 0) {
    console.log(`while: ${i}`);
    i--;
}
// do-while
// 블럭을 실행한 다음에 조건이 맞는지 안맞는지 검사함. 블럭을 먼저 실행하고 싶을떄만 사용

do {
    console.log(`do while: ${i}`);
    i--;
} while( i > 0 )

//for문 (begin; condition; step)
//begin을 처음에 딱한번만 호출을 하고 그담에 블럭을 실행전에 컨디션이 맞는지아닌지 검사해야
for(i =3; i >0 ; i --) {
    console.log();
}
```

### quiz (break, coutinue)
- 0부터 10까지의 숫자중 짝수만 프린트해야함(continue사용)
```javascript
- for(let i=0; i < 11; i++) {
    if(i%2 !==0) {
        continue;
    } 
    console.log(`q1. ${i}`);
}
```
- 주어진범위 0부터 10까지를 루핑하는것을 작성하돼 숫자 8을 만나면 그만해야함
```javascript
    for(let i = 0; i < 11 ; i++) {
        if(i > 8 ){
            break;
        }
         console.log(`q2. ${i}`);
    }

```
